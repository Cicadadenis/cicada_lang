# 🦗 Cicada — Язык программирования на русском

**Cicada v2.0** — интерпретируемый язык программирования с синтаксисом на русском языке.  
Написан на Python. Подходит для обучения, скриптинга и быстрого прототипирования.

---

## Установка 
```bash
curl -sSL https://raw.githubusercontent.com/Cicadadenis/cicada_lang/refs/heads/main/install.sh | bash


## 🚀 Запуск

```bash
# Запустить файл
python cicada.py мой_скрипт.cicada

# Интерактивный режим (REPL)
python cicada.py
```

> Файлы программ должны иметь расширение `.cicada`

- Python 3.7+

## 📝 Синтаксис языка

### Переменные и вывод

```
пусть имя = "Алиса"
пусть возраст = 25
вывод "Привет,", имя
```

### Условия

```
если возраст >= 18
    вывод "Совершеннолетний"
иначе_если возраст >= 14
    вывод "Подросток"
иначе
    вывод "Ребёнок"
конец
```

### Циклы

```
# Цикл пока
пусть i = 1
пока i <= 5
    вывод i
    i += 1
конец

# Цикл для
пусть числа = список(1, 2, 3, 4, 5)
для н в числа
    вывод н * н
конец
```

### Функции

```
функция приветствие(имя)
    вернуть "Привет, " + имя + "!"
конец

вывод приветствие("Саша")
```

### Обработка ошибок

```
попытка
    пусть х = число("abc")
ошибка е
    вывод "Ошибка:", е
конец
```

### Импорт файлов

```
импорт "мой_модуль"
```

## 🧩 Примеры использования

### === СЛОВАРИ ===

```
вывод "=== СЛОВАРИ ==="
пусть d = словарь(имя="Алиса", возраст=25)
вывод "Ключи:", ключи(d)
вывод "Есть ключ 'имя'?", есть_ключ(d, "имя")
удалить_ключ(d, "возраст")
вывод "После удаления:", d
```

### === МНОЖЕСТВА ===

```
вывод "\n=== МНОЖЕСТВА ==="
пусть a = множество(1,2,3)
пусть b = множество(3,4,5)
вывод "Объединение:", объединение(a,b)
вывод "Пересечение:", пересечение(a,b)
вывод "Разность:", разность(a,b)
```

### === MAP / FILTER / REDUCE ===

```
вывод "\n=== MAP/FILTER/REDUCE ==="
пусть л = список(1,2,3,4,5)
вывод "map (x*2):", map(функция(x) вернуть x*2 конец, л)
вывод "filter (x%2==0):", filter(функция(x) вернуть чётный(x) конец, л)
вывод "reduce (сумма):", reduce(функция(a,b) вернуть a+b конец, л)
```

### === REGEX ===

```
вывод "\n=== REGEX ==="
пусть текст = "email: test@example.com"
вывод "regex_найти:", regex_найти("[a-z]+@[a-z]+\\.[a-z]+", текст)
вывод "regex_совпадает:", regex_совпадает("email.*", текст)
вывод "regex_заменить:", regex_заменить("test", "demo", текст)
```

### === ОЧЕРЕДЬ И СТЕК ===

```
вывод "\n=== ОЧЕРЕДЬ И СТЕК ==="
пусть q = очередь(1,2,3)
enqueue(q, 4)
вывод "Очередь после enqueue:", q
вывод "dequeue:", dequeue(q)

пусть s = стек(10,20)
push(s, 30)
вывод "Стек после push:", s
вывод "pop:", pop(s)
```

### === HTTP ===

```
вывод "\n=== HTTP ==="
пусть html = http_получить("https://httpbin.org/get")
вывод "HTTP GET результат:", длина(html), "символов"
```

> ⚠️ Требуется установленный пакет `requests`

### === ФОРМАТИРОВАНИЕ ===

```
вывод "\n=== ФОРМАТИРОВАНИЕ ==="

```

## 🤖 Telegram-бот на Cicada

Простой эхо-бот. Требуется токен от [@BotFather](https://t.me/BotFather).

```
# Простой Telegram-бот на Cicada

пусть токен = "ВАШ_ТОКЕН_БОТА"
пусть url = "https://api.telegram.org/bot" + токен

функция получить_обновления()
    вернуть json_список(http_получить(url + "/getUpdates"))
конец

функция отправить_сообщение(chat_id, текст)
    вернуть http_отправить(url + "/sendMessage", словарь(chat_id=chat_id, text=текст))
конец

вывод "Бот запущен. Ожидание сообщений..."

пока истина
    пусть updates = получить_обновления()
    для upd в updates["result"]
        пусть chat_id = upd["message"]["chat"]["id"]
        пусть текст = upd["message"]["text"]

        если текст == "/start"
            отправить_сообщение(chat_id, "Привет! Я бот на языке Cicada.")
        иначе
            отправить_сообщение(chat_id, "Ты написал: " + текст)
        конец
    конец
    сон(2)  # пауза 2 секунды
конец
```

## 📚 Справочник встроенных функций

### Типы и преобразования
| Функция | Описание |
|---|---|
| `число(x)` | Преобразовать в целое число |
| `дробь(x)` | Преобразовать в дробное число |
| `текст(x)` | Преобразовать в строку |
| `булево(x)` | Преобразовать в булево значение |
| `список(a, b, ...)` | Создать список |
| `тип(x)` | Получить тип значения |

### Математика
| Функция | Описание |
|---|---|
| `абс(x)` | Модуль числа |
| `корень(x)` | Квадратный корень |
| `степень(x, н)` | x в степени н |
| `округл(x, н)` | Округление |
| `макс(a, b)` | Максимум |
| `мин(a, b)` | Минимум |
| `сумма(список)` | Сумма элементов |
| `среднее(список)` | Среднее арифметическое |
| `чётный(x)` | Проверить чётность |
| `нечётный(x)` | Проверить нечётность |

### Строки
| Функция | Описание |
|---|---|
| `длина(x)` | Длина строки или списка |
| `верхний(x)` | ВЕРХНИЙ регистр |
| `нижний(x)` | нижний регистр |
| `разделить(x, с)` | Разбить строку |
| `соединить(с, список)` | Объединить список в строку |
| `заменить(x, а, б)` | Заменить подстроку |
| `содержит(x, с)` | Проверить вхождение |
| `найти(x, с)` | Индекс подстроки |
| `обратный(x)` | Перевернуть строку |

### Списки
| Функция | Описание |
|---|---|
| `добавить(л, x)` | Добавить в конец |
| `убрать(л, i)` | Удалить по индексу |
| `сортировать(л)` | Сортировать список |
| `диапазон(а, б, ш)` | Список чисел от а до б с шагом ш |
| `уникальные(л)` | Убрать дубликаты |
| `zip(л1, л2)` | Соединить два списка в пары |

### Словари
| Функция | Описание |
|---|---|
| `словарь(к=в, ...)` | Создать словарь |
| `ключи(d)` | Список ключей |
| `значения(d)` | Список значений |
| `есть_ключ(d, k)` | Проверить наличие ключа |
| `удалить_ключ(d, k)` | Удалить ключ |

### Множества
| Функция | Описание |
|---|---|
| `множество(a, b, ...)` | Создать множество |
| `объединение(a, b)` | Объединение (a \| b) |
| `пересечение(a, b)` | Пересечение (a & b) |
| `разность(a, b)` | Разность (a - b) |

### Функции высшего порядка
| Функция | Описание |
|---|---|
| `map(ф, л)` | Применить функцию к каждому элементу |
| `filter(ф, л)` | Отфильтровать список |
| `reduce(ф, л)` | Свернуть список |

### Регулярные выражения
| Функция | Описание |
|---|---|
| `regex_найти(п, с)` | Найти все совпадения |
| `regex_совпадает(п, с)` | Проверить совпадение |
| `regex_заменить(п, з, с)` | Заменить по паттерну |

### Очереди и стеки
| Функция | Описание |
|---|---|
| `очередь(a, b, ...)` | Создать очередь |
| `enqueue(q, x)` | Добавить в очередь |
| `dequeue(q)` | Извлечь из очереди |
| `стек(a, b, ...)` | Создать стек |
| `push(s, x)` | Положить на стек |
| `pop(s)` | Снять со стека |

### HTTP и JSON
| Функция | Описание |
|---|---|
| `http_получить(url)` | GET-запрос |
| `http_отправить(url, д)` | POST-запрос |
| `json_текст(x)` | Объект → JSON строка |
| `json_список(x)` | JSON строка → объект |

### Дата и время
| Функция | Описание |
|---|---|
| `сейчас()` | Текущие дата и время |
| `дата()` | Текущая дата |
| `время()` | Unix-время |
| `сон(с)` | Пауза в с секунд |
| `час()` / `минута()` / `секунда()` | Компоненты времени |

### Файлы и система
| Функция | Описание |
|---|---|
| `прочитать(путь)` | Прочитать файл |
| `записать(путь, д)` | Записать в файл |
| `дописать(путь, д)` | Дописать в файл |
| `существует(путь)` | Проверить существование |
| `удалить_файл(путь)` | Удалить файл |
| `оболочка(команда)` | Выполнить команду ОС |
